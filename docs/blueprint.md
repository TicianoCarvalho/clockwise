# **App Name**: ClockWise

## Core Features:

- Registro de Ponto: Serviço de autoatendimento para os funcionários registrarem o ponto (entrada e saída) via PC ou celular.
- Painel de Administração: Painel de administração para gerenciar empresas, filiais, funcionários e horários.
- Gerenciamento de Empresas e Filiais: Configuração e gerenciamento de perfis de empresas e locais de filiais.
- Gerenciamento de Funcionários: Gerenciamento abrangente de perfis de funcionários.
- Gerenciamento de Horários e Turnos: Criação de horários de trabalho e turnos rotativos.
- Rastreamento de Horas: Rastreamento em tempo real das horas trabalhadas, ausências e horas extras.
- Relatório de Anomalias: Uma ferramenta de IA analisa os dados de registro de ponto em relação ao horário de trabalho. Com base em quaisquer incompatibilidades, esta ferramenta decide se deve enviar um relatório resumido de possíveis discrepâncias ao gerente apropriado.

## Style Guidelines:

- Cor primária: HSL 219, 71%, 46% (RGB Hex: #3D84F7). O azul brilhante evoca confiabilidade e profissionalismo, adequado para controle de tempo.
- Cor de fundo: HSL 219, 20%, 96% (RGB Hex: #F0F4FF). Um leve tom de azul cria um pano de fundo calmo e consistente.
- Cor de destaque: HSL 189, 67%, 46% (RGB Hex: #27A79F). Uma cor que se inclina para o verde-azulado complementa o azul, proporcionando contraste.
- Emparelhamento de fontes: 'Inter' (sans-serif) para títulos e corpo do texto para manter uma sensação moderna, neutra e objetiva em todo o aplicativo.
- Use ícones limpos e modernos para representar ações e pontos de dados.
- Um layout limpo e intuitivo com navegação clara.
- Transições e animações sutis para melhorar a experiência do usuário.
